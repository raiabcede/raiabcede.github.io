Privacy Policy

This extension is designed to protect your browser session by locking access to open tabs using a password. Your privacy is a core priority.

Data Collection

This extension does not collect, transmit, sell, or share any personal data. No information is sent to external servers or third parties.

Local Data Storage

All data is stored locally on your device using Chrome's chrome.storage.local, including:

  • A hashed password (SHA-256)
  • A hashed secret key for recovery
  • Lock state (locked/unlocked)
  • Temporarily stored tab URLs while the browser is locked
  • Failed login attempt count and timestamps

No data is stored in plain text.

Data Usage

Stored data is used only to:

  • Lock and unlock the browser
  • Prevent unauthorized access to tabs and settings
  • Restore tabs after unlocking
  • Monitor failed access attempts for security purposes

Data Retention

  • All data remains on your device
  • Tab URLs are deleted immediately after unlocking
  • All stored data is removed when the extension is uninstalled

No Tracking or Analytics

This extension:

  • Does not use analytics
  • Does not track browsing activity
  • Does not read website content or form data

Contact

If you have questions about this privacy policy, you may contact the developer via the Chrome Web Store listing.
